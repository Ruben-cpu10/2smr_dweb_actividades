Guía Completa de Etiquetas HTML con Ejemplos

Este documento explica el uso de las etiquetas HTML más comunes con ejemplos claros.

Estructura Semántica (Ejemplo Corregido)

El HTML semántico utiliza etiquetas que describen el significado de su contenido. Esto es crucial para la accesibilidad (lectores de pantalla) y el SEO (motores de búsqueda).

En lugar de un diagrama confuso, aquí tienes un ejemplo de código real que muestra cómo se estructuran estas etiquetas dentro del <body>.

Ejemplo de código de una maqueta web

<body>

    <!-- 1. CABECERA: Arriba del todo. Suele tener el logo y el menú. -->
    <header>
        <h1>Logo de Mi Sitio Web</h1>
        <nav>
            <ul>
                <li><a href="/">Inicio</a></li>
                <li><a href="/blog">Blog</a></li>
                <li><a href="/contacto">Contacto</a></li>
            </ul>
        </nav>
    </header>

    <!-- 2. CONTENIDO PRINCIPAL: Lo más importante de esta página. -->
    <main>
        <article>
            <h2>Título del Artículo Principal</h2>
            <p>Este es el contenido del artículo...</p>
            
            <!-- Una sección dentro del artículo -->
            <section>
                <h3>Comentarios</h3>
                <p>El primer comentario del post.</p>
            </section>
        </article>
    </main>

    <!-- 3. BARRA LATERAL: Contenido relacionado pero no principal. -->
    <aside>
        <h3>Enlaces Relacionados</h3>
        <ul>
            <li><a href="...">Otro artículo</a></li>
            <li><a href="...">Publicidad</a></li>
        </ul>
    </aside>

    <!-- 4. PIE DE PÁGINA: Al final de todo. Copyright, enlaces legales. -->
    <footer>
        <p>© 2025 Mi Sitio Web. Todos los derechos reservados.</p>
    </footer>

</body>

¿Dónde se coloca cada etiqueta? (Maqueta visual)

Imagina que la página web es un periódico:

  • <header>: Es la mancheta de arriba del todo (el nombre del periódico, el logo y la fecha).
  • <nav>: Es el índice de secciones (Deportes, Internacional, Cultura), que suele ir en la cabecera o justo debajo.
  • <main>: Es la noticia principal y más grande, que ocupa el centro de la portada.
  • <aside>: Es una columna lateral más pequeña, con noticias breves, el tiempo o publicidad. (En CSS se colocaría al lado del <main>).
  • <article>: Es cada noticia individual. Si tu <main> es un blog, tendría varios <article>.
  • <section>: Son partes dentro de una noticia, como "Antecedentes", "Desarrollo" o "Comentarios".
  • <footer>: Es el pie de imprenta al final de la página (la dirección, el copyright, enlaces legales).

<strong> y <b> - Negrita

<strong> se usa para indicar que un texto es muy importante. <b> solo pone el texto en negrita por estilo, sin darle importancia.

Ejemplo 1 (strong):

Advertencia: No desenchufar el equipo durante la actualización.

<em> y <i> - Cursiva

<em> se usa para dar énfasis a una palabra. <i> se usa para palabras en otro idioma, nombres técnicos o de barcos, solo por estilo.

Ejemplo 1 (em):

Estás totalmente equivocado.

Ejemplo 2 (i):

La expresión latina carpe diem significa "aprovecha el momento".

Citas

<blockquote> - Bloque de Cita

Se usa para citas largas que ocupan su propio párrafo. El navegador suele añadirle un margen.

Ejemplo 1:

La imaginación es más importante que el conocimiento. El conocimiento es limitado, mientras que la imaginación abarca el mundo entero.

Ejemplo 2:

Esto es una cita larga sacada de un libro, y por eso ocupa su propio espacio en la página.

<q> - Cita corta

Para citas cortas que van dentro de un párrafo. El navegador añade las comillas automáticamente.

Ejemplo 1:

Como dijo mi abuelo, nunca es tarde si la dicha es buena.

Ejemplo 2:

Ella me miró y me dijo no puedo creerlo.

<cite> - Citar una obra

Se usa para nombrar el título de una obra (un libro, una película, una canción...).

Ejemplo 1:

Mi libro favorito es El Señor de los Anillos.

Ejemplo 2:

La famosa pintura El Grito fue pintada por Edvard Munch.

Listas

<ul> y <ol> - Listas

<ul> es una lista desordenada (con puntitos). <ol> es una lista ordenada (con números). <li> es cada elemento de la lista.

Ejemplo 1: Lista de la compra (desordenada).

  • Leche
  • Pan
  • Manzanas

Ejemplo 2: Pasos a seguir (ordenada).

  1. Abrir la puerta
  2. Encender la luz
  3. Cerrar la puerta

<u>, <del>, <ins>, <s> - Edición y Estilo

<u> subraya. <del> tacha texto eliminado. <ins> subraya texto insertado. <s> tacha texto que ya no es correcto.

Ejemplo 1: Corrección de un texto

<code> y <pre> - Código

<code> se usa para una línea corta de código. <pre> para un bloque de código, ya que respeta los espacios y saltos de línea.
